# Welcome/Signup/Login Page

This repository contains a welcome/signup/login page template for a website. The page is designed using HTML, CSS, and JavaScript to provide a user-friendly interface for signing up and logging in.

## Features

- Sign up: Users can enter their name, mobile number, email, gender, and date of birth to create an account.
- Login: Existing users can log in using their credentials.
- Responsive Design: The page is optimized for various screen sizes and devices.
- SEO-friendly: The HTML structure and meta tags are optimized for search engine optimization.
- Validation: Input fields are validated to ensure correct data entry.
- Styling: The page is styled using CSS to create an attractive and intuitive user interface.
- Interaction: JavaScript is used to handle form submission and display relevant messages.

## Usage

1. Clone the repository to your local machine.
2. Open the `index.html` file in a web browser.
3. Fill in the required details in the signup form and click the "Submit" button to create an account.
4. Existing users can enter their login credentials and click the "Login" button to access their account.

## File Structure

- `index.html`: The main HTML file containing the welcome/signup/login page structure.
- `styles.css`: The CSS file for styling the page.
- `script.js`: The JavaScript file for handling form submission and interaction.
- `images/`: A directory containing any images used in the page.
